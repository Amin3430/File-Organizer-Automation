# File Organizer Automation  

A desktop application built with **Python**, **Tkinter GUI**, **JSON**, **OS**, and **Shutil** libraries for automating file organization. This tool helps users sort and move files into categorized folders (e.g., Documents, Images, Videos, Music, etc.) based on file extensions.  

---

## 🚀 Features  
- Simple **Tkinter GUI** for user interaction  
- Organize files into categories (Documents, Images, Music, Videos, Others)  
- Uses **JSON configuration** for extension-to-folder mapping  
- **os** and **shutil** libraries for file/folder operations  
- Supports drag-and-drop or folder path input  
- Real-time file organization with progress display  
- Cross-platform support (Windows/Linux/Mac)
